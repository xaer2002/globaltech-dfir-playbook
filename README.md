overview

This repository includes a digital forensics and incident response (DFIR) playbook for responding to a targeted ransomware attack.
The project covers the entire incident response lifecycle, forensic evidence processes, and containment/recovery strategies.

content

dfir-playbook.md - Key Incident Response Procedures

project-report.md - formal write-up of the project

user-manual.md - Quick guide for responders

Chain-of-Custody-Template.md - Evidence Documentation Template

Artifacts/- example screenshots, ransom notes and IOC files

Diagram/- IR and Forensics Workflow Diagram

how to use

Start with dfir-playbook.md for the full feedback process.

Use the chain of custody template when collecting evidence.

Review project-reports for academic presentations.

Use the user manual as a step-by-step operating guide.

ethical use

No actual systems, data or malware were accessed.
